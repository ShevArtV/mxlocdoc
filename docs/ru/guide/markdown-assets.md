# Markdown и ассеты

Markdown рендерится на сервере через Parsedown с включённым safe mode. Raw HTML экранируется renderer-ом.

## Ссылки

Относительные ссылки на Markdown-файлы открываются внутри manager UI mxLocDoc:

```markdown
[Установка](guide/setup.md)
```

Внешние ссылки остаются обычными ссылками браузера.

## Картинки и ассеты

Относительные картинки отдаются через защищённый connector:

```markdown
![Диаграмма](images/diagram.svg)
```

mxLocDoc проверяет путь, расширение и размер файла перед отдачей ассета. Запрос не может выйти за пределы настроенного корня документации.
