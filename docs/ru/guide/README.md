# Руководство

Этот раздел описывает рабочий цикл mxLocDoc: установку, настройки, навигацию, Markdown-файлы, локальные ассеты, поиск, очистку кэша и многоязычные корни документации.

Рекомендуемый подход: хранить публичную документацию в репозиторной папке `docs/` и перед сборкой пакета зеркалировать те же файлы в `core/components/mxlocdoc/docs/`. Так публичная документация и просмотрщик в менеджере MODX остаются синхронными.
