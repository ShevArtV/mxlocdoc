# Языки

mxLocDoc поддерживает мультиязычную документацию через языковые папки внутри `mxlocdoc.docs_path`.

Пример:

```text
docs/
├── en/
│   ├── _sidebar.json
│   └── README.md
└── ru/
    ├── _sidebar.json
    └── README.md
```

Если найдено больше одной языковой папки, manager UI показывает переключатель языка.

По умолчанию выбирается язык manager-а, если такая папка существует. Если совпадения нет, mxLocDoc выбирает первый доступный язык.

Если язык один, переключатель скрывается.
