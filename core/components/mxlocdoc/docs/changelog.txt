mxLocDoc Changelog
==================

0.1.0-pl
--------

- Initial MODX 2 package release.
- Add manager documentation viewer with three-column layout, breadcrumbs, collapsible navigation and page table of contents.
- Add system settings, setting lexicons and runtime option normalization.
- Add `[[+corePath]]` support and core-relative paths for `mxlocdoc.docs_path`.
- Add secure filesystem services, document and asset processors.
- Add navigation builder with `_sidebar.json`/`mxlocdoc.json` manifest support and filesystem fallback.
- Add safe Markdown renderer with protected relative asset URLs.
- Add live Markdown search with cached filesystem index in MODX cache.
- Add manager cache clear plugin for mxLocDoc search cache invalidation.
- Add multilingual documentation roots and manager language selector.
- Add publishable English and Russian documentation.
